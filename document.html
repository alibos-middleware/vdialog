引用

vdialog 是一款基于 jQuery 的免费开源插件，jQuery 最低版本要求：1.7.1++

首先，在页面的 <head> 标签内，引入 vdialog 的样式文件：

<link rel="stylesheet" href="./css/vdialog.css">

根据自己的需要，用下面三种方式引入 vdialog：

常规引用：

在页面的 <head> 标签内，引入 jQuery 及 vdialog：

<script src="./js/lib/jquery-1.10.2.js"></script>
<script src="./js/lib/vdialog.js"></script>

模块化：For sea.js / require.js

define(function(require, exports, module) {
  var vdialog = require('lib/vdialog');
  vdialog.alert('Hello, vdialog for CMD/AMD.');
});

 title Boolean | String对话框标题

  • 属性
  • 方法

默认情况下，对话框的标题为“提示信息”，若要显示为自定义的标题，请修改该属性为对应的字符串：

vdialog({
  title: '欢迎'
});

运行

若要隐藏标题栏，请设置该属性为 false

vdialog({
  title: false,
  content: '这是一个没有标题的 vdialog'
});

运行

你还可以通过 title() 方法来读取或者设置标题

var d = vdialog({
  title: '欢迎'
});
// 改变 title
d.title('快看，我变了');
// 隐藏 title
d.title(false);
// 显示 title
d.title(true).title('快看，我又变了');
// 获取 title
alert(d.title());

 type String 对话框类型（图标）

  • 属性
  • 方法

通过 type 可以设置对话框的类型，通常表现为图标显示不一样，可用值有：

  • alert
  • success
  • error
  • confirm
vdialog({
  type: 'alert',
  title: '欢迎',
  content: '我是一个优雅的 HTML5 对话框',
  modal: true,
  ok: true,
  cancel: true
});

运行

下面分别展示各种图标样式

var d = vdialog({
  title: '欢迎',
  content: '我是一个优雅的 HTML5 对话框',
  modal: true,
  ok: true,
  cancel: true
});
d.type('alert').content('我是提示信息！');
d.type('success').content('恭喜：操作成功！');
d.type('error').content('操作失败，服务器忙！');
d.type('confirm').content('你确定要删除该条记录吗？');

运行

除了通过设置 type 属性、调用 type 方法来设置对话框样式之外，还可以通过 vdialog.alert()、vdialog.success()、vdialog.error()、vdialog.confirm() 方法来实现，详情请阅读 其他

 returnValue Any 对话框关闭或退出时的返回值

  • 属性

给内容中的某个 表单元素 设置 data-returnable="true" 属性，可以告诉对话框当关闭对话框时的返回值，对话框只返回设置了该属性的表单元素值

vdialog({
  title: '欢迎',
  content: '您的姓名：<input type="text" data-returnable="true" />',
  ok: true,
  cancel: true,
  close: function(){
    alert(this.returnValue);
  }
});

运行

由于 close 是一内置事件（参考：事件），因此下面的代码，也能达到相同的功效：

var d = vdialog(...);
d.on('close', function(){
  alert(this.returnValue);
});

 show method 显示对话框

  • 属性
  • 方法
  • 事件

通过 show 方法可以显示一个对话框

vdialog({
  title: '欢迎',
  content: '我是一个优雅的 HTML5 对话框',
  ok: true,
  cancel: true
}).show();

运行

默认情况下，第一次创建的对话框不需要调用 show() 方法即可显示，所以该方法经常会省略。

在一个对话框隐藏的情境下，调用 show() 方法会显示该对话框。

若为 show() 方法传入参数 anchor，即代表该对话框将会在 DOM anchor 所在的位置下方出现

希望在本链接下面出现一个对话框 运行下面代码，将会在本链接下方出现对话框

vdialog({
  title: '欢迎',
  content: '我是一个优雅的 HTML5 对话框',
  ok: true,
  cancel: true
}).show(document.getElementById('method-show-mark'));

运行

 hide method 隐藏（但不销毁）对话框

  • 属性
  • 方法
  • 事件

通过 hide 方法可以隐藏一个对话框，不同于 close 方法，hide 操作后的对话框还在文档中，只是变成不可见而已，而 close 操作过的对话框将会从文档中彻底销毁

 on method 监听事件

  • 事件

通过 on，可以监听各种事件，常见的事件有：

  • show
  • hide
  • close
  • cancel

下面是一个监听 close 事件的例子：

var d = vdialog({
  title: '欢迎',
  content: '我是一个优雅的 HTML5 对话框',
  ok: true,
  cancel: true,
  modal: true
});
d.on('close', function(){
  alert('你关闭了对话框');
});

运行

从上面的演示可以看到，事件监听和前面涉及到的一些方法是类似的，例如上面的事件监听可以通过方法来做到：

var d = vdialog({
  title: '欢迎',
  content: '我是一个优雅的 HTML5 对话框',
  ok: true,
  cancel: true,
  modal: true
});
d.close(function(){
  alert('你关闭了对话框');
});

运行

 emit method 执行指定事件

  • 事件

on 方法除了能够监听常规的事件之外，还可以创建一个自定义事件监听器：

var d = vdialog(...);
d.on('myEvent', function(){
  alert('这是一个自己定义的事件 myEvent');
});

通过 emit，可以执行正在监听的事件：

下面是一个执行监听事件的例子：

var d = vdialog({
  title: '欢迎',
  content: '我是一个优雅的 HTML5 对话框',
  ok: function(){
    alert(this.content().html());
    this.emit('myEvent');
  }
});
d.on('myEvent', function(){
  alert('这是一个自己定义的事件 myEvent');
});
